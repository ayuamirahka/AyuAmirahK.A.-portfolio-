## 📋 Deskripsi

Website portofolio pribadi yang dirancang sebagai media presentasi diri secara digital untuk mendukung proses lamaran magang dan kerja sama proyek di bidang **Product Management**.

Website ini dibangun sebagai tugas individu mata kuliah **Desain Antarmuka Pengguna**, Program Studi Sistem Informasi, Fakultas Ilmu Komputer, Universitas Brawijaya, Semester Genap 2025/2026.

---

## 🗂️ Struktur Halaman

| Section | Deskripsi |
|---|---|
| **Navbar** | Logo AA., navigasi anchor link, tombol Hire Me |
| **Hero** | Nama, foto profil, tagline, tombol CTA |
| **About** | Narasi personal dan 4 value card |
| **Projects** | 3 kartu proyek dengan deskripsi dan tools |
| **Experience** | Timeline 3 pengalaman organisasi |
| **Skills** | Statistik angka dan 3 kelompok keahlian |
| **Education** | Kartu riwayat pendidikan Universitas Brawijaya |
| **Contact** | Email, GitHub, dan LinkedIn |
| **Footer** | Copyright dan tagline |

---

## ✨ Prinsip UI/UX yang Diterapkan

| Prinsip | Implementasi |
|---|---|
| **Konsistensi Desain** | Palet warna warm white + magenta, font Poppins konsisten di seluruh halaman |
| **Navigasi Jelas** | Fixed navbar, anchor link, hamburger menu di mobile |
| **Layout Rapi** | Grid CSS responsif, spacing seragam, hierarki visual jelas |
| **Tipografi** | Poppins 800 heading, 400 body, 300 deskripsi |
| **Warna Harmonis** | Warm white (#FAFAF8) + dark magenta (#8b2252) |
| **Responsif** | Breakpoint 640px (mobile) dan 900px (tablet) |
| **Feedback Visual** | Hover effect pada card, button, chip, dan link |
| **Aksesibilitas** | Semantic HTML5, aria-label, aria-expanded |
| **Scroll Animation** | IntersectionObserver fade-up per section |

## 🎨 Design System

### Color Palette
| Token | Hex | Penggunaan |
|---|---|---|
| Primary Background | `#FAFAF8` | Latar belakang utama |
| Card Surface | `#FFFFFF` | Latar belakang card |
| Elevated Surface | `#F3F0F8` | Section alternatif |
| Primary Accent | `#8b2252` | Tombol, label, aksen |
| Accent Soft | `#c96b92` | Teks italic, highlight |
| Text Primary | `#1C1C1C` | Heading dan teks utama |
| Text Muted | `#6B6566` | Deskripsi dan caption |

| Style | Font | Weight | Size |
|---|---|---|---|
| Hero Name | Poppins | 800 | 88px |

| Card Title | Poppins | 600 | 20px |
| Body | Poppins | 400 | 15px |
| Description | Poppins | 300 | 14px |

## 🚀 Cara Menjalankan Lokal

```bash
# Clone repository
git clone https://github.com/ayuamirahka/AyuAmirahK.A.-portfolio-.git

# Buka folder
cd AyuAmirahK.A.-portfolio-

# Buka di browser langsung
open index.html

# Atau gunakan Live Server di VS Code
# Klik kanan index.html → Open with Live Server
```

Tidak memerlukan build tool atau dependensi tambahan.

---

## 🌐 Deployment

Website di-deploy menggunakan **Vercel** dengan konfigurasi auto-deploy dari branch `main`. Setiap perubahan yang di-push ke GitHub akan otomatis ter-deploy dalam hitungan detik.

```
Repository GitHub → Push ke main → Auto Deploy → Live di Vercel
```

**Live URL:** https://ayu-amirah-k-a-portfolio.vercel.app/
